# Metal Strength 💪

A cross-platform (Android + iOS) fitness app that tells you **how strong you are**
from what you lift — your estimated 1-rep max, strength tier (Beginner → Elite),
population percentile, bodyweight ratio, and powerlifting scores (Wilks / DOTS /
IPF GL).

Built with **Flutter + Dart**. Fully offline — every formula and standard is
bundled. All strength logic is implemented from the public-domain / published
formulas in [`refrence`](refrence).

---

## ✨ Features

- **Accounts** — create an account or **Continue with Google** (Firebase Auth),
  plus a guest mode. See [SETUP.md](SETUP.md) to wire your Firebase project.
- **Landing screen** — a hero + one tap into the catalog, with a language switch
  and a donation button.
- **English & French** — pick the language on the landing screen; it's persisted.
- **Illustrated figures** — every lift shows a drawn human-figure performing the
  movement (no photos, no real person), grouped by movement pattern.
- **Easy exercise picker** — search + filter by equipment (barbell, dumbbell,
  kettlebell, machine, cable, bodyweight).
- **Single, industry-standard 1RM** — every estimate uses the **Epley** formula
  (no confusing formula picker). The big-3 lifts can still switch to a
  **powerlifting total** for Wilks / Wilks-2 / DOTS / IPF GL scoring.
- **Rich result** — animated tier gauge, percentile, bodyweight ratio, the gap to
  the next tier, the **world record** for that lift (men's or women's, by the
  user's sex), and **bullet-point tips** to get stronger.
- **Bodyweight movements** (pull ups, push ups, dips…) are rated by **rep count**.
- **Profile-aware** — sex, bodyweight, optional age (age-adjusted standards),
  kg/lb units. Saved between launches.

## 🚀 Getting started

> The repo ships the Dart source + assets. The platform folders
> (`android/`, `ios/`) are generated by Flutter on your machine.

1. **Install Flutter** (3.19+): https://docs.flutter.dev/get-started/install
   then `flutter doctor` until it's all green (needs Android Studio for the
   Android SDK + an emulator or a USB device).

2. **Generate the platform folders** (run once, inside the project — this keeps
   the existing `lib/`, `assets/`, `pubspec.yaml` and only adds what's missing):

   ```bash
   flutter create --org com.metalstrength --project-name metal_strength \
     --platforms=android,ios .
   ```

3. **Install packages & run:**

   ```bash
   flutter pub get
   flutter run
   ```

   The app boots straight into **guest mode** — no backend required. To enable
   real accounts / Google sign-in and the donation link, follow
   **[SETUP.md](SETUP.md)** (Firebase config, SHA-1 / URL scheme, donation URL).

4. **Run the tests / static analysis:**

   ```bash
   flutter test
   flutter analyze
   ```

## 🧮 The math

All formulas live in `lib/core/formulas/` and come straight from
[`refrence`](refrence):

- **1RM estimators** — §2
- **Strength tiers & percentiles** — §3, §4
- **Wilks / Wilks-2 / DOTS / IPF GL** — §5
- **Exercise catalog & ratio anchors** — §6
- **Age adjustment** — §6

Standards are ratio-based estimates. To upgrade the big-3 lifts to real
population percentiles, generate lookup tables from
[OpenPowerlifting.org](https://data.openpowerlifting.org) (public domain) per §7.
